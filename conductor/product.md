# Product Definition: Intervals Agent

## Vision
To provide actionable insights into race performance for an amateur cyclist by aggregating data from multiple sources (Intervals.icu, Xert, MyFitnessPal) and analyzing correlations between pre-race wellness/training metrics and race-day performance.

## Key Features
- **Data Aggregation:** Fetch activities and wellness data from Intervals.icu API.
- **Performance Scoring:** Calculate a unified "Performance Score" based on Speed, W/kg, Efficiency, and Cardiac Drift.
- **Correlation Analysis:** Identify which metrics (e.g., Sleep, HRV, Training Load) strongly predict good race performances.
- **Xert Integration:** Ingest .TCX files from Xert to extract advanced power and heart rate metrics.
- **Reporting:** Generate CSV reports and visual dashboards (scatter plots, correlation charts).
